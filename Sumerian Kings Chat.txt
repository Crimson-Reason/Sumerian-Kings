fzmwml: assume the kings are moving at relativistic speeds and in the same spaceship

GitHub Copilot: If all the kings are moving together at the same relativistic speed (in the same spaceship), then **they all experience the same time dilation**. This means their *experienced* lifetimes (`t0`) would be the same, but their *reign lengths* (as measured by people on Earth) would all be stretched by the same Lorentz factor.

**What changes in your code:**
- There is only **one velocity** for all kings.
- You can use the *longest reign* (or any reign) to solve for the spaceship's velocity, given `t0 = 40` years and `reign` (Earth years).

**Example:**
Suppose you want all kings to have experienced 40 years, and the *longest reign* is 1500 years (Etana).  
The time dilation formula is:
```
t0 = reign * sqrt(1 - v^2/c^2)
```
Solve for `v`:
```
v = c * sqrt(1 - (t0/reign)^2)
```
You only need to compute this **once**.

**Modified code:**

````python
import numpy as np
import matplotlib.pyplot as plt

king_names = [
    # ...existing king names...
]

reigns = np.array([
    # ...existing reigns...
])  # in Earth years

t0 = 40  # years (proper time experienced by each king)
c = 1

# Assume all kings are on the same spaceship, moving at the same velocity
# Use the maximum reign to calculate the required velocity
max_reign = reigns.max()
v = c * np.sqrt(1 - (t0 / max_reign)**2)

print(f"All kings must travel at v = {v:.4f} c to experience 40 years while the longest reign on Earth is {max_reign} years.")

# Calculate the experienced time for each king at this velocity
lorentz_factor = 1 / np.sqrt(1 - (v/c)**2)
experienced_times = reigns / lorentz_factor

# Print a table
print(f"{'King':<20} {'Reign':>6} {'Experienced (years)':>20}")
print("-" * 48)
for name, reign, exp_time in zip(king_names, reigns, experienced_times):
    print(f"{name:<20} {reign:>6} {exp_time:>20.2f}")

# Optional: Plot experienced time for each king
plt.figure(figsize=(10, 5))
plt.bar(king_names, experienced_times, color='steelblue')
plt.ylabel('Experienced Time (years)')
plt.title(f'Experienced Time of Sumerian Kings (All at v = {v:.4f}c)')
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()
````

**Summary:**  
- All kings have the same velocity.
- Their *experienced* reigns are much shorter than the legendary Earth years, but all are proportional.
- The plot shows how much time each king would have *felt* pass, if they all traveled at the same relativistic speed.

fzmwml: write a python code to find the speed of each king as a fraction of the speed of light

GitHub Copilot: Here is a function that calculates the speed of each king as a fraction of the speed of light, given their reign and the assumed proper lifetime (`t0`). It returns a list of tuples with the king's name, reign, and inferred velocity:

````python
import numpy as np

def king_speeds(king_names, reigns, t0=40):
    """
    Returns a list of (king, reign, velocity as fraction of c) tuples.
    """
    c = 1
    reigns = np.array(reigns)
    # Avoid division by zero or sqrt of negative
    velocities = np.where(
        reigns > t0,
        c * np.sqrt(1 - (t0 / reigns) ** 2),
        0.0
    )
    return list(zip(king_names, reigns, velocities))

# Example usage:
results = king_speeds(king_names, reigns, t0=40)
for king, reign, v in results:
    print(f"{king:<20} {reign:>6} {v:>10.4f}")
````

This will print each king's name, reign, and their inferred speed as a fraction of the speed of light.
